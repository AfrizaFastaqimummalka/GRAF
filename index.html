Graf Sirkuit Cₙ — Siklus
Graf sirkuit Cₙ adalah graf tertutup dengan n simpul dan n sisi yang membentuk sebuah siklus. Bilangan kromatiknya bergantung pada ganjil/genapnya n.
Aturan Bilangan Kromatik Siklus:
χ(Cₙ) = 2 jika n GENAP  |  χ(Cₙ) = 3 jika n GANJIL Alasan: Pada siklus genap, kita bisa bergantian warna 1-2-1-2. Pada siklus ganjil, simpul terakhir selalu bertetangga dengan simpul awal yang sama warna — butuh warna ketiga.

Graf Lengkap Kₙ
Graf lengkap Kₙ adalah graf dimana setiap pasang simpul terhubung langsung. Setiap simpul bertetangga dengan semua simpul lain, sehingga butuh warna berbeda untuk setiap simpul.
Bilangan Kromatik Graf Lengkap: χ(Kₙ) = n Setiap simpul harus berwarna berbeda karena semua saling bertetangga. Derajat setiap simpul = n−1, sehingga χ = Δ+1 (kasus pengecualian Teorema Brooks).

χ(K₅) = 5 — setiap simpul beda warna
χ(K₅) = 5

Graf Lengkap K₆

χ(K₆) = 6 — polinom kromatik: k(k-1)(k-2)(k-3)(k-4)(k-5)
χ(K₆) = 6
Graf Bipartit
Graf bipartit adalah graf yang simpul-simpulnya dapat dibagi menjadi dua himpunan tidak kosong sehingga setiap sisi menghubungkan simpul dari kelompok yang berbeda. χ(G) = 2 untuk semua graf bipartit.
Teorema Graf Bipartit: Graf G bipartit ⟺ χ(G) = 2 ⟺ G tidak mengandung siklus ganjil Teorema König: untuk pewarnaan sisi graf bipartit, χ'(G) = Δ(G) — selalu Kelas 1.

Graf Bipartit Lengkap K₂,₃

χ = 2 — dua kelompok simpul, sisi hanya antar kelompok
χ(K₂,₃) = 2

Graf Bipartit Lengkap K₃,₃

χ = 2 — Δ = 3, χ'(G) = Δ = 3 (Teorema König)
χ(K₃,₃) = 2 · χ'(K₃,₃) = 3

Graf Bipartit K₂,₄

χ = 2 — Δ = 4, χ'(G) = 4 (Kelas 1)
χ(K₂,₄) = 2

Graf Bipartit Tidak Lengkap

χ = 2 — sisi tidak harus semua terhubung
χ = 2
Graf Roda Wₙ
Graf Roda Wₙ terdiri dari satu simpul pusat yang terhubung ke semua simpul siklus Cₙ di sekelilingnya. Bilangan kromatiknya: χ(Wₙ) = χ(Cₙ) + 1 karena simpul pusat butuh warna ekstra.
Aturan Bilangan Kromatik Graf Roda: χ(Wₙ) = 3 jika n GENAP  |  χ(Wₙ) = 4 jika n GANJIL Intuisi: Simpul pusat P harus beda warna dari SEMUA simpul luar. Total = χ(Cₙ) + 1 warna ekstra untuk pusat.

Algoritma Welch-Powell — Langkah Demi Langkah
Algoritma Welch-Powell: mulai dari simpul berderajat terbesar, beri warna seefisien mungkin. Klik "Langkah Berikutnya" untuk melihat proses pewarnaan secara bertahap.
Langkah Welch-Powell: 1. Hitung derajat setiap simpul. 2. Urutkan dari terbesar ke terkecil. 3. Beri Warna-1 ke simpul pertama dan semua yang tidak bertetangga dengannya. 4. Ulangi dengan Warna-2, Warna-3... hingga semua simpul terwarnai. Heuristik — tidak selalu optimal, tapi efisien untuk masalah nyata.

Teorema Utama — Visualisasi Perbandingan
Tiga teorema utama dalam teori kromatik dan perbandingan visualnya.

Teorema Brooks (1941)

χ(G) ≤ Δ(G)

Untuk semua graf terhubung, KECUALI siklus ganjil (Cₙ) dan graf lengkap (Kₙ) dimana χ = Δ+1.

Teorema Vizing (1964)

Δ ≤ χ'(G) ≤ Δ+1

Pewarnaan SISI: hanya 2 kemungkinan. Kelas 1: χ'=Δ (bipartit). Kelas 2: χ'=Δ+1 (Petersen).

Teorema König (1916)

χ'(G) = Δ(G)

Graf bipartit selalu Kelas 1. Pewarnaan sisi optimal = Δ warna.

Teorema 4 Warna (1976)

χ(G) ≤ 4

Setiap graf planar (bisa digambar tanpa sisi bersilangan) cukup 4 warna. Dibuktikan komputer.

Perbandingan Graf Brooks — χ ≤ Δ vs χ = Δ+1:
Graf di bawah menunjukkan K₄ (pengecualian Brooks: χ = Δ+1 = 4) vs graf biasa (χ ≤ Δ = 3).

K₄ — Pengecualian Brooks

Δ = 3, χ = 4 = Δ+1 (bukan ≤ Δ!)
χ = Δ+1 (pengecualian)

C₅ — Pengecualian Brooks Siklus Ganjil

Δ = 2, χ = 3 = Δ+1 (pengecualian siklus ganjil)
χ = Δ+1 (pengecualian)

Graf Petersen — Kelas 2 Vizing

Δ = 3, χ'(G) = 4 = Δ+1 (Kelas 2 Vizing)
χ'(G) = Δ+1 (Kelas 2)

Graf Bipartit K₃,₃ — Kelas 1 König

Δ = 3, χ'(G) = 3 = Δ (Kelas 1, Teorema König)
χ'(G) = Δ (Kelas 1)
Polinom Kromatik χ_G(k)
Polinom kromatik χ_G(k) menghitung jumlah CARA mewarnai graf G dengan tepat k warna secara sah. Hubungan: χ(G) = nilai k terkecil sehingga χ_G(k) ≠ 0.

Graf Diskret K̄ₙ (tanpa sisi)

χ_G(k) = kⁿ

Bebas pilih warna apapun untuk setiap simpul. Contoh n=3: k³ cara.

Graf Lengkap Kₙ

k(k−1)(k−2)···(k−n+1)

Setiap simpul butuh warna berbeda. K₃: k(k−1)(k−2) cara.

Pohon T (n simpul)

χ_G(k) = k(k−1)ⁿ⁻¹

Pohon dengan 4 simpul: k(k−1)³ cara mewarnai secara sah.

Siklus Cₙ

(k−1)ⁿ + (−1)ⁿ(k−1)

C₃: (k−1)³ − (k−1) = (k−1)(k²−3k+3). Coba k=3 → 6 cara.

Contoh Perhitungan Polinom Kromatik C₃ (Segitiga):
χ_G(k) = (k−1)³ + (−1)³(k−1) = (k−1)³ − (k−1) = (k−1)[(k−1)² − 1] = (k−1)(k−2)(k)
Cek: k=3 → 2×1×3 = 6 cara. k=2 → 1×0×2 = 0 cara (tidak bisa). ✓ Bilangan kromatik = 3.

Pohon T (4 simpul) — χ_G(k) = k(k−1)³

k=2: 2×1 = 2 cara. k=3: 3×8 = 24 cara
χ(T₄) = 2

Siklus C₃ — χ_G(k) = k(k−1)(k−2)

k=3: 6 cara. k=4: 24 cara mewarnai secara sah
χ(C₃) = 3

Graf Diskret K̄₃ — χ_G(k) = k³

Tanpa sisi: k=2 → 8 cara, k=3 → 27 cara
χ(K̄₃) = 1

Graf Lengkap K₄ — χ_G(k) = k(k−1)(k−2)(k−3)

k=4: 4! = 24 cara. k=3: 0 cara (tidak cukup warna)
χ(K₄) = 4
Aplikasi Nyata — Penjadwalan Perkuliahan
Teori kromatik digunakan untuk menjadwalkan perkuliahan: setiap mata kuliah = simpul, konflik (dosen/mahasiswa sama) = sisi, warna = slot waktu, χ(G) = jumlah slot minimum.
Model Graf Konflik: Buat sisi antara dua matkul jika mereka tidak bisa dijadwalkan bersamaan (dosen sama / mahasiswa sama). Lalu warnai dengan Welch-Powell → setiap warna = satu slot waktu. χ(G) = jumlah slot waktu MINIMUM yang dibutuhkan
